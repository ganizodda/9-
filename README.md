# Beauty Salon Landing Page

## 📖 Overview
This project is a clean and modern landing page design for a **Beauty Salon**.  
It highlights natural hair treatments and provides an elegant user experience with a focus on simplicity and professionalism.

---

## ✨ Features
- **Navigation Bar**: Includes logo `beautysalon.` and menu options:
  - Início
  - Sobre
  - Serviços
  - Depoimentos
  - Contato
- **Hero Section**:
  - Left side: Promotional text emphasizing natural hair health.
  - Right side: Image of a stylist blow-drying a client’s hair in a modern salon.
  - Call-to-action button: **Agendar um horário** (Book an appointment).
- **Modern Aesthetic**:
  - Neutral colors and large windows for a fresh, natural vibe.
  - Clean typography and layout for readability.

---

## 🛠️ Tech Stack
- **HTML5** for structure
- **CSS3** for styling (flexbox and responsive design)
- **JavaScript** (optional) for interactivity

---

## 🎯 Purpose
The landing page is designed to:
- Promote natural hair treatments.
- Provide a professional online presence for a São Paulo-based salon.
- Encourage visitors to schedule appointments easily.

---

## 🚀 How to Use
1. Clone or download the project files.
2. Open `index.html` in your browser.
3. Customize text, images, and styles to fit your salon’s branding.
